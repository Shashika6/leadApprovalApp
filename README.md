## Lead Approval app

Stack used :- 

T3 Stack

* Typescript
* ReactJS
* TailwindCSS
* NextJS
* Prisma
* SQLlite

## What does it do ?

This web app is in charge of 

- Approving or rejecting team leads. 
- Requesting for amendtments.
- Cycling through all of the leads.

## How to run ? 

- npm install
- npm run dev
- Open http://localhost:3000 to view it in your browser
